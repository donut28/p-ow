# Privacy Policy
**Last Updated:** January 10, 2026

## 1. Introduction
This Privacy Policy explains how Project Overwatch ("POW", "we", "us", or "our") collects, uses, and protects your personal information when you use our Service.

## 2. Information We Collect
### 2.1 Information You Provide
| Data Type | Source | Purpose |
|-----------|--------|---------|
| **Roblox Account** | OAuth via Clerk | Username, user ID, avatar for identification |
| **Discord Account** | OAuth via Clerk | Username, user ID, avatar for notifications |
| **Service Usage** | User actions | Shift times, LOA requests, punishment records |

### 2.2 Information Collected Automatically
| Data Type | Source | Purpose |
|-----------|--------|---------|
| **Game Logs** | PRC API | Player joins/leaves, kills, commands from ERLC servers |
| **Analytics** | PostHog | Pageviews, feature usage, session data, and session recordings |
| **Security Logs** | Server | IP addresses, access patterns for security |

## 3. Third-Party Services
We use the following third-party services to operate POW:
- **Clerk**: Authentication and User Management.
- **PostHog**: Analytics and Session Recording (EU-hosted).
- **Discord**: Notifications and Bot integration.
- **Roblox/PRC API**: Game data integration.

### 3.5 Legal Basis for Data Processing (GDPR)
- **Contractual Necessity**: Processing account info and shift data to provide the Service.
- **Legitimate Interests**: Security logging (IP addresses) to prevent abuse and platform security.
- **Consent**: Enhanced analytics and persistent session recording via PostHog.

## 11. Cookies and Tracking
We use the following tracking technologies:

### Authentication Cookies
- **Purpose**: Session management and login persistence.
- **Provider**: Clerk.
- **Opt-out**: Not available (required for Service functionality).

### Analytics Tracking
- **Purpose**: Usage tracking, session recording, product improvement.
- **Provider**: PostHog (EU).
- **Accept Cookies**: Enables persistent tracking and long-term usage analysis.
- **Decline Cookies**: Enables **Cookieless Memory Mode**. No data is stored on your device. Sessions are reset upon page refresh.

You can manage your preferences at any time via the "Cookie Preferences" link in the dashboard footer.
