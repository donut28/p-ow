# POW Vision

Roblox moderation overlay with OCR player detection.

## Features

- **OCR Player Detection** - Press Alt+V to scan and detect player name
- **Player Lookup** - Instant player info from POW dashboard
- **Quick Actions** - Warn, Kick, Ban directly from overlay
- **Configurable Hotkeys** - Set your own key combinations
- **Transparent Overlay** - Works over Roblox (borderless fullscreen)
